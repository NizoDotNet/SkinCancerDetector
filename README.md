# Melanoma Skin Cancer Classification (Benign vs Malignant)

This repository contains a full-stack project for detecting **benign vs
malignant melanoma** from skin lesion images using a **Convolutional
Neural Network (CNN)**. It includes both server-side model inference and
a client-side interface for uploading images.\
